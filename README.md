## 🚀 How to Use

```bash
# Clone the repo
git clone https://github.com/YOUR_USERNAME/js-basics.git
cd js-basics

# Run any file with Node.js
node 01-variables-and-datatypes/variables.js
```

> No dependencies. Pure vanilla JavaScript. Node.js required for running locally.
